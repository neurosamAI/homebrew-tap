# Homebrew Tap for neurosam.AI

> Created by [Murry Jeong (comchangs)](https://github.com/comchangs) · Supported by [neurosam.AI](https://neurosam.ai)

## Install

```bash
brew install neurosamAI/tap/tow
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [tow](https://github.com/neurosamAI/tow-cli) | Lightweight, agentless deployment orchestrator |

## Update

```bash
brew upgrade tow
```
